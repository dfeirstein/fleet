# Reference library — best-in-class to study

The sites, apps, tools, and standards the three designers point to. Study these
when you need a concrete model — grouped by what each exemplifies and by context.
Research-before-drawing is rule #1: pull 3–6 of these before you open a canvas.

## Disproportionate product UI (the ~90/8/2 floor)
- **Vercel** — pinnacle of disproportionate product distribution (~90% black / 8%
  white / 2% red), 98% white light-mode backgrounds, elite landing presentation.
- **Linear** — neutral foundation, 99% white backgrounds, blue-purple functional
  accent, elite landing presentation.
- **Stripe** — top-tier SaaS cognitive-fluency benchmark: clear logical layouts,
  minimal color, high-contrast type.
- **Supabase** — monochromatic neutral layering + bright-green functional accent;
  pricing-page patterns.
- **Mercury** — 2% brand-tinted sidebar anchoring a 100% pure-white content area
  (subtle depth, zero borders).
- **Notion** — 100% white backgrounds, dark-cards-on-light; **cautionary** emoji
  aesthetic that fails in dense SaaS (study what NOT to carry over).

## First-impression / Halo-Effect benchmarks
- **Apple** — confident minimalism, single bold headlines, full-screen visuals,
  sequenced iMac scroll + z-index occlusion reveals, balanced text blocks, smooth
  micro-interactions, zero cognitive load.
- **Figma** — making complex software feel simple/approachable; reference for
  kerning + iOS corner smoothing + nudge settings.

## Color-system exemplars
- **Lemon Squeezy** — 60-30-10 with **WHITE as the 10% accent** on purple/yellow.
- **Airbnb** — gold standard for **search → filter → browse** intent; brand pink
  with adjusted UI-state pinks.
- **Headspace** — ultra-light tinted orange background splash.
- **Mailchimp** — bright yellow primary + complementary turquoise.
- **GitHub dark (#040D21) / Studio Rubric light (#FFF8F0)** — tinted-neutral
  background exemplars (never pure #FFF/#000).

## Premium / luxury whitespace
- **Hermès / Bottega Veneta** — extreme-whitespace luxury exemplars ('we don't need
  to shout'); products float on white with massive margins.
- **Stryds** — Apple-product aesthetic: pure-black voids, bottom pill nav,
  brand-aura over conversion.
- **London AV** — premium 'blink-test' example (dark mode, full-bleed cinematic
  imagery). **ShopNow** — the amateur counter-example (purple-blue gradient, thin
  vector icons, centered generic).

## Engineered editorial / motion (gate behind flagship + damped)
- **Drive Capital** — text-as-mask windowing + scroll clipping.
- **Starfades International** — expanding-container color-wipe text (stacked clipped
  layers).
- **Miro 'The Ways We Work'** — editorial collage data viz, self-drawing SVG
  (stroke-dashoffset + IntersectionObserver), multi-speed parallax, intentional
  grid-breaks.
- **Jitter** — bento restraint, stark tri-color palette, smart mobile focal-crop.
- **Gleec** — horizontal→vertical scroll handoff tied to 3D Z-axis with damped
  physics.
- **Sketch** — interactive draggable product hero (show-don't-tell).
- **YouTube Tarot** — 3D Z-axis depth, fan-out card hover.

## Research + capture sources (use BEFORE drawing)
- **Mobbin** — reverse-engineer best-in-class real-world product UI/UX patterns.
- **Lapa.ninja** — curated live high-converting landing pages.
- **Typewolf** — trending high-end font pairings / type in the wild.
- **WhatTheFont** — reverse-engineer the exact typeface from a screenshot.
- **Behance / Dribbble (top artists, micro-interactions only) / Awwwards
  (aesthetics only)** — never lift their page STRUCTURE; Unsplash / Pexels for stock.

## Type
- **Google Fonts / Fontshare** — Inter, DM Sans, Geist, Plus Jakarta Sans,
  Montserrat, Poppins (sans); Merriweather, Lora, Source Serif Pro (serif).
- **Hurme Geometric Sans 3 + Inter** — premium display/body pairing reference.
- Other cited families: SF Pro, Axiforma, Plus Jakarta Sans, Unbounded.

## Icons
- **Phosphor Icons / Lucide** — preferred monochromatic SVG icon libraries.

## Color + contrast math
- **OKLCH / HSB color models** — perceptually uniform color + theming math (chart
  palettes locking L ~0.5682 / C ~0.1136 stepping hue 25–30; accent ramps; the
  L−0.03/C+0.02/hue→brand tinting formula).
- **Tailwind CSS color scale** — gray-900/600/400, 50/500 light, 950/300 dark as
  implementation vocabulary.
- **Coolors.co** — palette generation.
- **WCAG 2.2** — contrast standard (4.5:1 text, 7:1 premium body, ~3:1 large
  headers).

## Motion + build + instrumentation
- **GSAP ScrollTrigger** (pinning, xPercent, power3.out) + **IntersectionObserver**
  — engineered, damped scroll motion (and scroll-triggered SVG draw-in).
- **GA4 / heatmaps** — launch-day instrumentation for drop-off + A/B iteration.
- **Squarespace / Webflow / Wix** — strict auto-snapping 12-column build platforms;
  **Builder.io** — Figma-to-production visual editor.
- **InVision / Claude Design / Markups.AI / Loomia** — prototyping + wireframe-to-
  prototype tooling.
