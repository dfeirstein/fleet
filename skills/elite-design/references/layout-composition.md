# Layout & composition

The rules that govern **eye-flow, grid, and proportion** — how the eye enters a
layout, where it goes, and how long it stays. Load this when you are composing a
page/section/screen and need to control attention, not just style elements. **This
is a floor — the brand brief sets direction and wins any conflict.**

## Six levels of eye-flow (compose up this ladder)
- **1 · Basic direction** — explicit pointers: arrows, a subject's gaze, wayfinding
  lines. The eye follows where the content literally points.
- **2 · Hierarchy-driven flow** — visual **WEIGHT** is a gravitational pull; the eye
  falls **large → medium → small**. Build the route by scale/contrast, not arrows.
- **3 · Layered paths** — ONE main route + smaller micro-routes; balance so a
  secondary path never overpowers the primary (or the eye loses the thread).
- **4 · Implied motion** — repetition, progressive scaling, tapering, or frozen
  action creates direction with no moving parts.
- **5 · Flow disruption** — one intentional interrupter (e.g. a square rotated
  **45°** in a grid of upright cards) the eye hits, pauses on, then re-engages.
  Exactly one — a second disruptor cancels the first.
- **6 · Temporal flow** — control **WHEN** the eye arrives and how long it lingers:
  rhythm of **Impact → Linger/slow → Release/pause** (whitespace is the pause).

## L.I.F.T. (audit any composition against four letters)
- **Leverage point** — the single most important element, won by **scale / contrast
  / isolation**. Push back competitors (dim, shrink, space them away) so one thing
  clearly wins. If two elements tie, you have no leverage point.
- **Internal rhythm** — eye choreography: predictable **alignment** for calm +
  deliberate **contrast** for emphasis. Repetition sets the beat; one break is the
  accent.
- **Friction & flow** — **good friction** grabs (an intense focal point); **flow**
  (whitespace) lets the eye breathe between beats. A layout needs both, sequenced.
- **Transferability** — it must survive **shrinking, dark/light bg, mobile / print /
  motion**. If the composition only works at one size on one background, it fails.

## Grid systems (pick the grid to the content)
- **Column** — vertical columns for text/macro-layout. **Modular** — rows × columns
  = cells, for galleries/cards. **Compound** — column macro-layout with a modular
  sub-grid inside a block (e.g. Amazon: column page, modular product gallery).
  **Isometric** — a diamond dot-grid for 3D/technical illustration.
- **Spanning = dominance** — a block spanning **more columns** reads as more
  important. Width is a hierarchy lever, not just a fit constraint.
- **Macro whitespace** — leave whole **rows/columns empty** for premium breathing
  room. **Micro whitespace** — subdivide the grid (**3×4 → 6×6**) for tight, even
  inter-element spacing. Use both: empty macro cells, tight micro cells.

## Five grid-composition principles (score a built grid against these)
The five levers that separate a real composition from a hero-over-image. Score each
**Present / Weak / Absent**; an elite grid rebuild (Process Phase 3) lands all five.
1. **Extreme scale contrast** — one element makes a huge jump; everything else recedes
   (eye-flow level 2 + the L.I.F.T. leverage point).
2. **Active whitespace** — empty shape used as structure, not leftover (macro whitespace;
   the Release beat of temporal flow).
3. **Cell-spanning macro-blocks** — custom merged cells, not a uniform card grid
   (spanning = dominance).
4. **Shared vertical axes** — eyebrow / headline / subhead / card locked to common column
   lines (internal-rhythm alignment + the baseline grid).
5. **Exactly one intentional grid-break** — a single expressive bleed off-axis, tied to real
   meaning; a second cancels the first (eye-flow level 5, flow disruption).

## Proportion & alignment (start from math, then tweak)
- **Baseline grid** — set the headline as a **multiple of the body's leading** (e.g.
  **28pt** headline on **14pt** body, a 2× step) so type lines align across columns.
- **Golden ratio ≈ 1 : 1.618** — a *starting* proportion for splits/sidebars/crops,
  not a law. Set it, then adjust to the content — don't obey it rigidly.
- **Rule of thirds** — divide the frame into **9 parts** (2 horizontal + 2 vertical
  lines); place the key element on an **intersection** (off-center) for tension over
  a dead-center, static placement.

→ **See** `references/visuals/proportion.png` — rule of thirds, golden ratio, and the baseline grid, all to scale.

## Friction — good vs bad (the make-or-break)
- **Good friction** = one intense focal point + clean, readable type around it; the
  eye is grabbed, then guided. This is the goal of every composition.
- **Bad friction** = visual noise: overlapping/clipped text, **multiple competing
  focal points**, too many fonts/colors, elements added only to fill space. The eye
  stalls and bounces — it reads as amateur. Cut until one thing wins.
