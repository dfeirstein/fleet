Six levels of eye-flow — compose up the ladder

The index that ties the individual eye-flow visuals together. Each rung adds control over where the eye goes and how long it stays — climb from 1 to 6, and reach for the level the composition needs. The right column points to the visual that demonstrates it.

1
Basic direction
Explicit pointers — arrows, a subject’s gaze, wayfinding lines. The eye follows where the content literally points.
seeframe-basic-direction.png
2
Hierarchy-driven flow
Visual weight is gravity; the eye falls large → medium → small. Build the route by scale & contrast, not arrows.
seevisual-weight-flow.png
3
Layered paths
One main route + smaller micro-routes; balance them so a secondary path never overpowers the primary.
seeframe-layered-paths.png
4
Implied motion
Repetition, progressive scaling, tapering, or frozen action — direction with no moving parts.
seeno single frame — read the rule
5
Flow disruption
One intentional interrupter — a square rotated 45° in a grid of upright cards — the eye hits, pauses on, re-engages. Exactly one.
seeframe-flow-disruption.png
6
Temporal flow
Control WHEN the eye arrives and how long it lingers: Impact → Linger → Release (whitespace is the pause).
seeframe-temporal-flow.png
