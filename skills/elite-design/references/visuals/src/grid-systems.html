Grid systems — pick the grid to the content

Four grids, four jobs. Draw the line-work first, then place content into it. Width is a hierarchy lever, not just a fit constraint: spanning more columns reads as more important.

Column
Vertical columns for text / macro-layout. Spanning = dominance.
Modular
Rows × columns = even cells, for galleries & card systems.
Compound
A column page with a modular sub-grid inside one block (e.g. Amazon).
Isometric
A diamond dot-grid for 3D / technical illustration.
Macro whitespace
Leave whole rows / cols empty
Premium breathing room comes from emptying entire grid tracks, not shrinking gaps.
Micro whitespace
Subdivide 3×4 → 6×6
Finer cells give tight, even inter-element spacing. Use both: empty macro cells, tight micro cells.
